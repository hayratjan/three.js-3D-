
# 用Three.js创建一个实时交互的3D粒子系统。 
### 要求： 
- 1.通过摄像头检测双手张合控制粒子群的缩放与扩散 
- 2. 提供UI面板可选择爱心/花朵/土星/佛像/烟花等模^ 
- 3.支持颜色选择器调整粒子颜色 
- 4.粒子需实时响应手势变化 
- 5.界面简洁现代，包含全屏控制按钮


## 功能特点
### 🎨 粒子形态
- 爱心 - 经典心形曲线
- 花朵 - 五瓣花朵形状
- 土星 - 球体+光环结构
- 佛像 - 头部+身体+莲花座
- 烟花 - 多点爆发效果
- 银河 - 螺旋星系结构
### 🖐️ 手势控制
- 张开双手 → 粒子扩散
- 握拳 → 粒子收缩
- 实时响应，平滑过渡
### 🎛️ 控制面板
- 粒子数量调节 (1000-20000)
- 粒子大小调节
- 旋转速度控制
- 颜色选择器 + 预设颜色
- 手势感应数值显示
### 📊 信息显示
- 实时 FPS 显示
- 当前粒子数量
- 当前形态名称
### 🔧 其他功能
- 全屏模式切换
- 摄像头开关控制
- 一键重置所有参数
